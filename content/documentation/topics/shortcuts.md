_If you are a MacOS user, pretend `ctrl` is actually `cmd`_

- `ctrl+Enter` Send Request
- `ctrl+P` Switch Requests
- `ctrl+N` New Request
- `ctrl+D` Duplicate Request
- `ctrl+L` Focus URL Bar
- `ctrl+E` Show Environment Editor
- `ctrl+K` Show Cookie Editor
- `ctrl+,` Show Settings
- `ctrl++` Zoom In
- `ctrl+-` Zoom Out
- `ctrl+0` Reset Zoom
